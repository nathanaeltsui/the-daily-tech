Microsoft Ends Support for Windows 10: Offers Free Extended Security Updates

  • James Pennington
  • October 28, 2025

Ever since the official release of Windows 11 back on October 5 2021, when Microsoft offered users a free upgrade from Windows 10, the eventual demise of Windows 10 has always been lingering at the back of users' minds. However, that day has finally arrived when Microsoft announced on October 14 2025 that Windows 10 has been moved to its "end of life" phase, where it will continue to work without receiving security updates.

This has sparked significant distress among Windows users, with users complaining on forum sites such as Reddit and Quora. As of October 2025, Windows 10 still holds a market share of approximately 40% among Windows users, a number rapidly dropping following the end of support. However, not all hope is lost as Microsoft has offered a solution for those not quite ready to switch over to Windows 11.

Besides upgrading to Windows 11, or just purchasing a new PC with Windows 11 pre-installed, users can opt to sign up for the Extended Security Updates (ESU), allowing users to use Windows 10 for another year without security risks. Here are the steps to doing so:

Step 1: Ensure your PC is up to date

Check if your computer is up to date by going into Settings > System > About, then scroll down to see what version you're running. If not, make sure to install all the Windows 10 updates available.

Step 2: Use an administrator account for the process

Confirm that you are currently on an administrator account under Settings > Your Info, especially if the PC is shared between users. If not, be sure to sign in to the administrator account.

Step 3: Check for Windows 11 eligibility

Check if your computer is eligible for Windows 11. If an upgrade option is available, it's recommended to switch instead. If not, continue with the following steps.

Step 4: Enrol in Extended Security Updates

To enrol in ESU, navigate to Settings > Update & Security, then click the “Enroll Now” link. If there is an option to download Windows 11, do that instead.

Step 5: Choose enrollment method

Choose your enrollment method. The free option is to backup your PC settings using Microsoft's OneDrive service, which takes a bit of time. Once signed in, click “Add Device” to complete the enrollment.
